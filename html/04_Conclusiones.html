Qu es lo que ms contribuye a que existan fallecimientos en un accidente?

Para el estudio de lo que ms contribuye a que haya ms fallecimientos, hemos recurrido a la matriz de correlaciones que obtuvimos en el EDA.

Observamos como las variables con las que ms se correlaciona positivamente, es C_CONF, es decir cmo ocurri el accidente, y P_ISEV, que indica el grado de severidad. No obstante, estas variables no van a determinar qu factor influye en que haya ms fallecimientos.

Seguidamente, con menos correlacin destacan las variables C_SAFE,C_RCFG, C_TRAF. Con ello se puede observar como el elemento de seguridad juega un papel muy determinante a la hora de producirse fallecimientos o no. Adicionalmente, donde se produjo el accidente y la seal de trfico tambin determinan el nmero de fallecimientos.

Por ltimo tambin destacamos C_RALN y V_TYPE, que indican el tipo de carretera (curva ascendente, curva cerrada...) y el tipo de vehculo tambin juegan un papel determinante para que se produzcan fallecimientos o no. Todo ello, se explica porque no todos los vehculos estn hechos para todos los tipos de carreteras.
